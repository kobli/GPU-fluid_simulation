Semestrální projekt z předmětu B4M39GPU

Particle based fluid simulation on GPU

Aleš Koblížek

Zadání

Implement fluid simulation method called SPH first on CPU, then on GPU. Use an acceleration datastructure for neighbour search.

Ovládání

usage: ./demo [particleN [subdivisionN [boxSize [windowSize]]]]
Number of particles and number of subdivisions must be a power of two.

Keyboard controls:
esc - exit
r - restart simulation

At runtime, the following SPH coefficients can be adjusted (multiplied or divided by two) by pressing the listed keys:
S/s - simulation step
H/h - kernel radius
M/m - particle mass
K/k - pressure coefficient
U/u - viscosity coefficient

Afterwards, the current values will be printed to standard output.
